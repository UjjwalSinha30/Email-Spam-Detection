# SMS Spam Classifier

A Machine Learning web application that classifies SMS messages as Spam or Ham (Not Spam) using Natural Language Processing (NLP).

## Features
- Detects spam SMS messages
- NLP-based text preprocessing
- TF-IDF Vectorization
- Machine Learning classification
- Flask web application interface

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- NLTK

## Machine Learning Workflow
1. Text preprocessing
2. Tokenization
3. Stopword removal
4. TF-IDF Vectorization
5. Model training using Naive Bayes
6. Prediction through Flask app

